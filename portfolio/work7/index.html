Arduino Ejercicio Piano

Este es un ejercicio básico de Arduino UNO para probar sensores y códigos. En esta ocasión utilizo una interfaz virtual y doy indicaciones de la manera más básica para entender un código. Vamos a utilizar un módulo Digital.

Piano Buzzer

El piano buzzer es un buen ejercicio para que podamos entender el funcionamiento básico de Arduino. En general, todas las cosas funcionan con la misma lógica; definir los puertos que vamos a utilizar (pins), un “void setup” que es para declarar qué cosas funcionarán como input y output, y un “void loop” que va a ejecutar lo que le estemos pidiendo mediante programación.

Código
//Primero seteamos los pin que utilizaremos para conectar nuestros sensores
//El número antes del semicolon ";" corresponde al número de pin en nuestro módulo digital de arduino.

int but1 = 2;
int but2 = 3;
int but3 = 4;
int but4 = 5;
int but5 = 6;
int but6 = 7;
int but7 = 8;
int but8 = 9;

int buzzer = 13;

void setup()
{
  //Declaramos los pin de los botones como input.
  pinMode(but1,INPUT);
  pinMode(but2,INPUT);
  pinMode(but3,INPUT);
  pinMode(but4,INPUT);
  pinMode(but5,INPUT);
  pinMode(but6,INPUT);
  pinMode(but7,INPUT);
  pinMode(but8,INPUT);
  
  //El buzzer recibirá la señal, por ende lo declaramos como output.
  pinMode(buzzer,OUTPUT);
  
}

void loop()
{
  //Declaramos el valor de los sensores para que suenen con notas distintas.
  //Como la señal que estamos utilizando es digital, varía solamente en 0 y 1.
  //1 es para que cuando hagamos contacto, suene. 0 Hará lo inverso.
  int b1 = digitalRead(but1);
  int b2 = digitalRead(but2);
  int b3 = digitalRead(but3);
  int b4 = digitalRead(but4);
  int b5 = digitalRead(but5);
  int b6 = digitalRead(but6);
  int b7 = digitalRead(but7);
  int b8 = digitalRead(but8);
  
  if( b1 == 1 ){
     tone(buzzer,300,100);
  }
    if( b2 == 1 ){
     tone(buzzer,400,100);
  }
    if( b3 == 1 ){
     tone(buzzer,500,100);
  }
    if( b4 == 1 ){
     tone(buzzer,600,100);
  }
    if( b5 == 1 ){
     tone(buzzer,700,100);
  }
    if( b6 == 1 ){
     tone(buzzer,800,100);
  }
    if( b7 == 1 ){
     tone(buzzer,900,100);
  }
    if( b8 == 1 ){
     tone(buzzer,1000,100);
  }
  
//Aquí seteamos el delay que tendra la nota al hacer el contacto
  delay(10);
    
}
